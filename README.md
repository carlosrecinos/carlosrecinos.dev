# carlosrecinos.dev

Personal portfolio — Carlos Recinos, Senior Full-Stack & Applied AI Engineer.

Single-page site. **Vite + React + TypeScript + Tailwind CSS v4.** No router, no backend, no CMS.
All content lives in typed data files under `src/data/`.

```bash
npm install
npm run dev      # http://localhost:5273
npm run build    # -> dist/
npm run preview
```

Deployed on Vercel at https://carlosrecinos.dev
